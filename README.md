# ES-Writer
## 概要
<h4>自身の経歴をあらかじめ入力しておくことにより、自身の経歴からAIでES生成します</h4>
<h4>拡張機能にすることにより、画面遷移なく、実際のページに自動的に書き込みます</h4>
<h4>経歴は変わらないのに同じような内容を毎回書くのが勿体ないと感じ、これらの作業をAIに任せることにより、開発などの有意義な時間を増やすことができます！</h4>
